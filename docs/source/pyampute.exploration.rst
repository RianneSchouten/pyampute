pyampute.exploration module
===========================

The ``pyampute.exploration`` module contains functionality for inspecting incomplete datasets. 

:class:`~pyampute.exploration.md_patterns.mdPatterns` displays all unique missing data patterns, which is useful for investigating whether there is structure (or a good explanation) for the occurrence of missing data. 

:class:`~pyampute.exploration.mcar_statistical_tests.McarTest` performs a statistical hypothesis test to evaluate whether it is likely that missingness patterns are Missing Completely At Random (MCAR).
